# Python学生信息管理系统

## 核心功能
- 登录注册
- 数据总览
- 学生信息管理
- 学生成绩管理
- 教师信息管理
- 考试信息管理
- 通知管理

## 适用场景
- 教育机构内部管理
- 学生档案维护
- 教师资源管理
- 考试安排与成绩统计
- 通知发布与信息传播

## 技术特点
- 使用Python Flask框架
- 前端技术包括HTML、CSS、JavaScript、jQuery
- 后端使用Python语言
- 数据库支持MySQL
- 开发环境兼容Pycharm、Navicat
- 系统环境要求Python 3.0以上、MySQL 8.0
